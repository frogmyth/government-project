# 🔄 정부지원사업 작업 가이드

> AI 에이전트(이쁜이)와 협업하여 정부지원사업 문서를 작성하는 워크플로우

---

## 📋 작업 순서

### 1단계: 사업 분석

```
1. 공고문 PDF 다운로드
2. AI에게 공고문 분석 요청
   - 지원 자격
   - 평가 기준
   - 제출 서류
   - 마감일
3. 이전 탈락 사유 분석 (있는 경우)
```

### 2단계: 전략별 초안 작성

```
4개 관점에서 각각 초안 작성:
- 기술중심: 기술력, 개발방법, 성능지표
- 시장중심: TAM/SAM/SOM, 경쟁사, 사업화
- ESG중심: 환경, 사회, 지배구조 효과
- 글로벌중심: 해외 진출, 수출 전략
```

### 3단계: 통합 및 최종본 작성

```
1. 4개 초안의 강점 통합
2. 서식에 맞춰 내용 정리
3. 복사-붙여넣기용 문서 생성
```

### 4단계: 제출

```
1. hwpx 서식에 내용 복사
2. 필수 첨부서류 준비
3. PMS 시스템 접수
```
