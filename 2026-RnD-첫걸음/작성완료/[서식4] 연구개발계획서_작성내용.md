4. **음성 명령 인터페이스**: 매장 직원용 음성 CMS 제어 (Whisper + NLU)
5. **AI 플래노그램 고도화**: 날씨 연동 수요 예측, Computer Vision 준수율 측정

## 연구개발 방법

### [연구방법]

◦ **1단계: AI 챗봇 CMS (2026.6~2026.12)**
1. sLLM 파인튜닝: Llama 3 8B + LoRA, 리테일 도메인 10만 데이터셋 학습
2. RAG 파이프라인 구축: CMS DB + 템플릿 DB 벡터 인덱싱
3. Tool Calling 개발: CMS API 자동 호출 (태그 배치, 콘텐츠 생성, 예약)
4. 대화 흐름 설계: Multi-turn 대화, 오류 복구, 확인 단계

◦ **2단계: AI 롤링 최적화 (2027.1~2027.6)**
1. 고객 행동 분석 모델: 시간대별 체류 시간, 클릭률 분석
2. 롤링 순서 최적화: 강화학습 기반 콘텐츠 배치 최적화
3. 자동 A/B 테스트: Multi-armed Bandit 알고리즘 적용
4. 위치별 맞춤 콘텐츠: 계산대/입구/냉장고 등 위치별 전략

◦ **3단계: 생성형 AI 고도화 (2027.7~2027.12)**
1. Out-painting: SAM + SDXL Inpainting으로 비정형 디스플레이 대응
2. 스타일 자동 추천: 카테고리 분류 → 스타일 매칭 모델
3. 일괄 생성: 상품 목록 → 배치 이미지/동영상 생성 파이프라인

◦ **4단계: 음성 인터페이스 & 플래노그램 (2028.1~2028.5)**
1. STT 고도화: Whisper Large-v3 + 리테일 용어 파인튜닝
2. NLU 개발: 의도 분류 + 슬롯 필링 (매장, 상품, 할인율 추출)
3. 플래노그램 AI 비전: YOLO 기반 상품 인식, 준수율 자동 측정
4. 날씨 연동: 기상청 API → 3일 후 최적 플래노그램 제안

# 1. 개발기술 개요

## 1-1. 개발기술의 필요성

### 1-1-1. 개발 배경

◦ **글로벌 ESL 시장 급성장**
- 2025년 32억 달러 → 2030년 87억 달러 (CAGR 22.1%)
- 아시아태평양 시장 2030년 31억 달러 (CAGR 23.0%)

◦ **기존 ESL의 한계**
- 단순 가격 표시 기능만 제공 (Pricer, SES-imagotag, Hanshow)
- AI 기반 마케팅 콘텐츠 자동 생성 기능 전무
- 배터리 교체 필요 (환경 문제, 유지보수 비용)

◦ **리테일 미디어 네트워크(RMN) 성장**
- 2025년 600억 달러 → 2028년 1,500억 달러 예상
- 매장 내 디지털 마케팅의 중요성 급증

### 1-1-2. 기술적 필요성

◦ **AI 콘텐츠 자동 생성 필요**
- 중소 리테일: 전문 디자이너 부재, 마케팅 콘텐츠 제작 어려움
- 대형 리테일: 수천 개 상품의 콘텐츠 제작 시간/비용 과다

◦ **비전문가 친화적 인터페이스 필요**
- 기존 CMS: 복잡한 메뉴 구조, 학습 곡선 높음

◦ **AI 기반 최적화 필요**
- 콘텐츠 롤링: 수동 설정 → AI 자동 최적화
- 플래노그램: 경험 기반 → 데이터 기반 배치

### 1-1-3. 경제적·사회적 필요성

◦ **ESG 경영 강화**
- 탄소중립 2050 정책: Battery-Free 솔루션 필요
- 종이 사용 감축: 연간 500톤 절감 가능

◦ **K-리테일테크 수출 기반**
- H마트 미국 88개점 진출 계획
- 동남아 리테일 시장 진출 기회
