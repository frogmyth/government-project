# 정부지원사업 계획서 작성 스킬

정부 R&D 지원사업 신청서 작성을 위한 AI 에이전트 스킬

---

## 주요 기능

### 1. 공고문 분석
- PDF 공고문에서 핵심 정보 추출
- 평가 기준, 지원 자격, 제출 서류 정리
- 마감일, 지원금 규모 파악

### 2. 탈락 사유 분석
- 이전 신청 탈락 사유 분석
- 각 지적 사항별 보완 방안 제시
- 보완 내용 초안에 반영

### 3. 전략별 초안 작성
4가지 관점에서 초안 작성:
- **기술중심**: 핵심 기술, 개발 방법, 성능 지표
- **시장중심**: TAM/SAM/SOM, 경쟁사, 사업화 전략
- **ESG중심**: 환경, 사회, 지배구조 효과
- **글로벌중심**: 해외 진출, 수출 전략

### 4. 경쟁사 분석
- 글로벌/국내 경쟁사 조사
- URL, 매출, 핵심 기능, 가격 정리
- 차별점 분석

### 5. 통합 문서 생성
- 4개 초안의 강점 통합
- 서식에 맞춰 최종본 작성
- 복사-붙여넣기용 문서 생성

---

## 사용 예시

### 공고문 분석
```
사용자: 이 공고문 분석해줘
AI: [PDF 분석 후 요약]
- 사업명: ...
- 지원금: ...
- 마감일: ...
- 평가기준: ...
```

### 초안 작성
```
사용자: 기술중심 초안 작성해줘
AI: [기술력 강조 초안 생성]
```

### 통합 문서
```
사용자: 4개 초안 통합해서 최종본 만들어줘
AI: [강점 추출 및 통합]
```

## 핵심 팁

### 디딤돌 탈락 사유 대응
| 탈락 지적 | 보완 방법 |
|----------|----------|
| 경쟁사 비교분석 미흡 | 5개 이상 경쟁사 비교표 |
| 개발 방법 부족 | 단계별 알고리즘/코드 예시 |
| 성능지표 근거 없음 | 벤치마크 출처 명시 |
| TAM/SAM/SOM 없음 | 시장 규모 + 출처 |
| 매출 목표 근거 없음 | 판매처/수량/단가 산출 |

### 필수 포함 요소
- 핵심 차별점 (세계 최초/유일 강조)
- 정량적 성능 지표 (목표 + 측정 방법)
- 특허 출원 계획
- 기대 효과 (기술적/경제적/사회적)
