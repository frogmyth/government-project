## 🚀 빠른 시작

### 1. 클론

```bash
git clone https://github.com/frogmyth/government-project.git
cd government-project
```

### 2. 최신 문서 확인

```bash
# 마스터 문서 열기
open "2026-RnD-첫걸음/📋_정부지원사업_종합정리.md"

# 최종 초안 확인

```

### 3. 수정 후 푸시

```bash
git add .

git push
```
