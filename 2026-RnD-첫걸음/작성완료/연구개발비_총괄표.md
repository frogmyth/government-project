## 규정 적합성 체크

| 규정 | 산출 | 판정 |
|------|------|:----:|
| 인건비 ≤ 총연구개발비 50% | 100,000/200,000 = 50.0% | ✅ |
| 신규인건비 ≤ 인건비합계 50% | 50,000/100,000 = 50.0% | ✅ |
| 기존직원 = 현물, 신규직원 = 현금 | 기존 50,000 현물 / 신규 50,000 현금 | ✅ |
| 연구수당 ≤ 총연구개발비 2.5% | 5,000/200,000 = 2.5% | ✅ |
| 간접비 = 0 (영리기관) | 0 | ✅ |
| 연구활동비 현금만 | 현금만 계상 | ✅ |
| 정부지원금 ≤ 1.5억/년 | 150,000천원 = 1.5억 | ✅ |
| 기업부담금 0.5억/년 | 50,000천원 = 0.5억 | ✅ |

## 1차↔2차 년도 차이점

- **연구시설·장비비**: 1차(30,000) → 2차(20,000) — 초기 장비 집중 투자
- **연구재료비**: 1차(25,000) → 2차(30,000) — 2차년도 실험·테스트 확대
- **연구활동비**: 1차(40,000) → 2차(45,000) — 2차년도 검증·출장·전문가활용 증가
