| **중국어** | Whisper-ZH + jieba | 중문 간체/번체 | BERT-base-chinese | 중국 마트 관습 용어 |

**실시간 언어 감지 알고리즘:**
```python
def detect_language_and_route(audio_input, touch_coords, text_input):
    # 1. 오디오 기반 언어 감지 (100ms)
    lang_prob = whisper_language_detection(audio_input)
    
    # 2. 터치 패턴 기반 UI 언어 추정 (10ms)
    ui_lang = infer_ui_language(touch_coords, current_display_layout)
    
    # 3. 가중 평균으로 최종 언어 결정
    final_lang = weighted_vote([lang_prob, ui_lang], weights=[0.7, 0.3])
    
    # 4. 해당 언어 모델로 라우팅
    return route_to_language_model(final_lang, audio_input, touch_coords, text_input)
```

```python
async def generate_global_content(product_info, target_countries):
    tasks = []
    
    for country in target_countries:
        # 국가별 문화 맥락 로딩
        cultural_context = load_cultural_params(country)
        
        # 병렬 콘텐츠 생성
        task = asyncio.create_task(
            generate_localized_content(
                product_info=product_info,
                language=country_to_language[country],
                cultural_style=cultural_context,
                regulatory_compliance=get_regulations(country)
            )
        )
        tasks.append(task)
    
    # 모든 국가 버전 동시 생성 (30초 이내)
    results = await asyncio.gather(*tasks)
    return results
```
