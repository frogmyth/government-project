## 3. 제안 기능 — 🚀 신규 개발 우선순위

### 🥇 최우선 (Phase 1: 2026년)

#### **3.1 📸 AI 비전 기반 플래노그램 준수율 모니터링**
> **경쟁사**: SymphonyAI, Stoc, e-con Systems
> **논문**: Nature Scientific Reports (2025.12) - 대만 7-Eleven 7,000개 매장 적용

**기능 상세:**
```
[현장] Digital Rail 카메라 → 선반 이미지 촬영 (1시간 단위)
                          ↓
[Edge] YOLO v8 + ByteTrack → 상품 인식 + 위치 추출
                          ↓
[Cloud] 플래노그램 비교 → 준수율 계산 (목표: 95% 정확도)
                          ↓
[출력] 위반 사항 알림 + 자동 보정 제안
```

**기대 효과:**
- 수동 점검 시간 90% 절감
- 준수율 40% 향상 (Core-Mark 사례)
- 결품/오배치 실시간 감지

**AIsirius 차별점:**
- Digital Rail 디스플레이 내장 카메라 활용 (추가 하드웨어 불필요)
- 기존 CMS와 완전 통합

---

#### **3.2 🌤️ 날씨/계절 연동 수요 예측 플래노그램**
> **경쟁사**: RELEX, LEAFIO, Walmart
> **기술**: 기상청 API + ML 시계열 예측

**기능 상세:**
```
[입력] 기상청 API (3일 예보) + 과거 매출-날씨 상관관계
                          ↓
[분석] 날씨별 카테고리 수요 변화 학습
       - 비 오는 날: 우산 +300%, 아이스크림 -40%
       - 무더위: 음료 +150%, 라면 -20%
       - 한파: 즉석식품 +80%, 아이스크림 -60%
                          ↓
[출력] 3일 후 최적 플래노그램 자동 제안
       "금요일 폭염 예보 → 음료 골든존 확대, 라면 축소"
```

**기대 효과:**
- 날씨 연동 매출 15~25% 증가 (RELEX 사례)
- 계절 상품 재고 손실 30% 감소

**AIsirius 차별점:**
- 한국 기상청 API 네이티브 연동
- 편의점/마트 특화 카테고리 모델

---

#### **3.3 🏪 Store Clustering (매장 클러스터링)**
> **경쟁사**: Blue Yonder, Nielsen
> **기술**: K-Means, DBSCAN 클러스터링

**기능 상세:**
```
[입력] 매장 속성 데이터
       - 위치 (주거지/오피스/학교 근처)
       - 면적 (소형/중형/대형)
       - 고객층 (직장인/학생/가족)
       - 매출 패턴 (평일형/주말형)
                          ↓
[분석] 유사 매장 그룹화 (예: "오피스 상권 소형 편의점")
                          ↓
[출력] 클러스터별 표준 플래노그램 자동 생성
       "A그룹(오피스) → 도시락/커피 강화"
       "B그룹(학교) → 과자/음료 강화"
```

**기대 효과:**
- 플래노그램 작성 시간 50% 단축
- 신규 매장 즉시 최적 배치 적용

**AIsirius 차별점:**
- 한국 상권 데이터 학습 (배달앱 + 유동인구)
- 중소 리테일 맞춤 (대기업 솔루션 대비 저비용)

---

### 🥈 우선 (Phase 2: 2027년)

#### **3.4 🔮 신상품 성과 예측 배치**
> **경쟁사**: Antuit.ai, Nielsen
> **기술**: 유사 상품 매칭 + Transfer Learning

**기능 상세:**
```
[입력] 신상품 정보 (카테고리, 가격, 브랜드, 패키지)
                          ↓
[분석] 유사 기존 상품 검색 (임베딩 기반)
       → 유사 상품의 출시 초기 성과 분석
       → 예상 매출 등급 예측 (A/B/C/D)
                          ↓
[출력] 신상품 최적 배치 위치 제안
       "이 신상품은 기존 '콜라' 유사 → 골든존 배치 추천"
```

**기대 효과:**
- 신상품 실패율 40% 감소
- 출시 첫 주 매출 예측 정확도 80%+

---

#### **3.5 📊 Price Elasticity (가격 탄력성) 기반 배치**
> **경쟁사**: RELEX, dunnhumby
> **기술**: 가격-수요 상관관계 분석

**기능 상세:**
```
[입력] POS 데이터 (가격 변동 + 판매량)
                          ↓
[분석] 상품별 가격 탄력성 계산
       - 탄력적 상품: 가격 10% ↓ → 판매 30% ↑
       - 비탄력적 상품: 가격 10% ↓ → 판매 5% ↑
                          ↓
[출력] 탄력적 상품 → 프로모션 시 골든존 우선
       비탄력적 상품 → 하단 배치해도 매출 유지
```

**기대 효과:**
- 프로모션 ROI 25% 향상
- 할인 상품 최적 배치

---

#### **3.6 🛒 Cross-Selling 자동 인접 배치**
> **경쟁사**: LEAFIO, Blue Yonder
> **기술**: Association Rule Mining (Apriori)

**기능 상세:**
```
[입력] POS 장바구니 데이터
                          ↓
[분석] 동시 구매 패턴 분석 (Support, Confidence)
       - 라면 + 계란: 동시 구매율 35%
       - 맥주 + 안주류: 동시 구매율 60%
       - 빵 + 우유: 동시 구매율 45%
                          ↓
[출력] 인접 배치 자동 제안
       "맥주 진열대 옆에 오징어/땅콩 배치 추천"
```

**기대 효과:**
- 교차 판매 20% 증가
- 객단가 15% 향상

---

#### **3.7 📱 AR 현장 미리보기**
> **경쟁사**: 신규 (대부분 미지원)
> **기술**: ARKit/ARCore + 3D 렌더링

**기능 상세:**
```
[현장] 스마트폰으로 진열대 촬영
                          ↓
[AR 오버레이] AI 제안 배치를 AR로 시각화
             "이 상품을 여기로 이동하세요" 가이드
                          ↓
[확인] 배치 완료 후 사진 촬영 → 준수율 자동 체크
```

**기대 효과:**
- 배치 변경 작업 시간 50% 단축
- 교육 없이 누구나 정확한 배치 가능
